# Product Brief

Negotiation Mechanism Replay turns one sanitized negotiation event into a
typed report that teaches a reusable mechanism failure to portfolio repos.

## User

- Single technical operator.
- Needs reusable incentive structure extracted from a confidential event.
- Needs each lesson attached to a concrete repo slug and file-level change.

## Job

- Capture a redacted summary.
- Map actor incentives, leakage, and implied equilibrium.
- Attach the mechanism to specific portfolio repo changes.
- Block unsafe artifacts before commit.

## v0.1 Scope

- Local Python CLI.
- Markdown report with JSON front matter.
- Repo index and mechanism taxonomy.
- Redaction, schema, voice, and spec gates.
- One checked-in JSONL report artifact under `reports/`.

## Exclusions

- Raw negotiation storage.
- Real-time negotiation assistance.
- Multi-user workflow.
- Generic mechanism education without a named recipient repo.
