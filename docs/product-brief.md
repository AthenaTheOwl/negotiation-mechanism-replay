# Product Brief

Negotiation Mechanism Replay is a local report generator for sanitized
negotiation lessons. A user records one safe event, maps actor incentives,
names leakage, states the implied equilibrium, and assigns the same mechanism
failure to a concrete portfolio repo.

## User

- A single technical operator who handles recurring procurement, capacity, and
  escalation negotiations.
- The user wants the reusable mechanism, not the confidential facts.
- The user needs file-level portfolio changes, not broad advice.

## Job

- Convert one redacted event into one typed Markdown report.
- Block unsafe reports before commit.
- Keep each cross-application attached to a repo slug and one-sprint file
  change.

## v0.1 scope

- Local CLI.
- JSON-front-matter Markdown report.
- Repo index and mechanism taxonomy.
- Redaction, schema, spec, and voice gates.
- One checked-in report artifact.

## Out of scope

- Raw negotiation storage.
- Real-time negotiation assistance.
- Multi-user workflow.
- Embedding-based matching.
